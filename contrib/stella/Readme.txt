Graphical tool:
A full featured graphical tool can be found at
www.bigambiente.com/stella.
The source code is available via a git repository:
git://github.com/davidgraeff/stellacontrol.git

C-Only test tool:
In this folder a lighter c-only test program is present.
Open the file "stella_test.c" and change STELLA_PORT
and STELLA_IP according to your setup.
To comile just type in "./make", propably "nmake" in a
windows environment. (Check if you have gcc installed)
Help is available via "./stella_test --help"
